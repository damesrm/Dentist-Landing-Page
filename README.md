# Dental Landing Page

This project is a landing page developed for a dental clinic, with the purpose of presenting its services, building a professional online presence, and making patient contact easier.

## About the project

The idea behind this page is to work as a digital showcase for a dentist or dental clinic, gathering important information in a clear, organized, and visually appealing way.

The interface was designed to provide simple navigation, highlight services, present the professional in a trustworthy way, and encourage appointment scheduling.

## Features

- Presentation of the dentist or clinic
- Dental services section
- Modern and clean design
- Responsive layout for different devices
- Call-to-action buttons for scheduling or contact
- Organized and intuitive structure

## Technologies used

- HTML
- CSS
- JavaScript

## Purpose

This project was developed as part of my portfolio, with the goal of practicing the creation of professional, responsive landing pages for real business scenarios.
